Créer un Nouveau Dépôt sur GitHub

  • Auteur : ACOMDIR https://www.facebook.com/acomdir/.

Étape 1 : Créer un Dépôt sur GitHub

  1. Connectez-vous à votre compte GitHub.
  2. Cliquez sur le bouton New (nouveau) ou allez à https://github.com/new.
  3. Remplissez les informations suivantes :
    • Repository name : Le nom de votre dépôt (exemple : MonProjet).
    • Description : (optionnel) Ajoutez une courte description de votre projet.
    • Choisissez entre Public ou Private selon vos besoins.
    • Cochez Initialize this repository with a README pour ajouter un fichier README par défaut.
  4. Cliquez sur Create repository pour finaliser.

Étape 2 : Préparer le Dossier Local

Ouvrez l'Invite de commandes Windows et naviguez jusqu'au dossier contenant vos fichiers :

cd chemin\vers\votre\dossier

Étape 3 : Initialiser Git dans le Dossier Local

Si le dossier n'est pas déjà un dépôt Git, initialisez-le :

git init

Cela créera un sous-dossier .git pour suivre les modifications.

Étape 4 : Lier le Dépôt Local au Dépôt GitHub

Ajoutez l'URL du dépôt distant :

git remote add origin https://github.com/VotreNomUtilisateur/MonProjet.git

Étape 5 : Ajouter les Fichiers au Dépôt Local

Ajoutez tous les fichiers du dossier au suivi Git :

git add .

Pour ajouter des fichiers spécifiques, utilisez :

git add nom_du_fichier

Étape 6 : Créer un Commit

Enregistrez vos modifications avec un message descriptif :

git commit -m "Premier commit : ajout des fichiers du projet"

Étape 7 : Pousser les Modifications vers GitHub

Envoyez les fichiers et l'historique des modifications vers le dépôt distant :

git push -u origin main

Si la branche principale porte un autre nom (comme master), remplacez main par le nom correct.

Conseils Supplémentaires

  • Utilisez git status pour vérifier l'état du dépôt local.
  • Utilisez git pull origin main pour synchroniser votre dépôt local avec les dernières modifications du dépôt distant.
